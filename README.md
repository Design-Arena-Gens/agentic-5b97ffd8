# Nodi Verse Lead Radar

Curated intelligence dashboard for identifying high-value creative collaboration opportunities for **Nodi Verse**, the next-gen cinematic animation studio. Built with Next.js, Tailwind, and a structured data layer covering premium brands across fashion, tech, F&B, lifestyle, beauty, and entertainment.

## Features

- Dynamic lead directory with intelligent search and multi-filtering by industry, region, and platform.
- Rich brand dossiers outlining visual hooks, content opportunities, recommended deliverables, and outreach angles.
- Snapshot analytics covering industry and regional spread plus follower reach benchmarks.
- Responsive, deployment-ready UI designed for Vercel.

## Running Locally

First, run the development server:

```bash
npm run dev
```

Then visit [http://localhost:3000](http://localhost:3000).

## Deploying

The project is optimized for one-click production deployments on Vercel:

```bash
vercel deploy --prod
```

Ensure the `VERCEL_TOKEN` environment variable is available when running the command in CI or automated environments.
